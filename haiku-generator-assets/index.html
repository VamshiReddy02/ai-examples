This Haiku generator is a demonstration of how you can use Fermyon Serverless AI to easily make an AI-powered API. When you type in a topic it is sent to a Spin app running in the Fermyon Cloud and a haiku is generated with a pre-trained model.

Note that LLM's are not perfect and the haikus performed by this application is not guaranteed to perfectly follow the format of 5-7-5.

To get started type a topic below and press enter.
